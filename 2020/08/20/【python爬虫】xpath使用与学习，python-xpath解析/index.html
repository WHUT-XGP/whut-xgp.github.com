xpath路径学习

绝对路径起始于正斜杠( / )，而相对路径不会这样。

  • /：下一层级 /html/body/div
  • //: 多个层级 /html//div或者//div（表示任意位置开始定位）
  • . :当前层级
  • .. :当前节点的父节点
  • nodeName 选取该节点的所有子节点
  • @:获取属性 可以使用 + - * div(除法) mod or and >= <= = !=
  • //div[@class=”song”] 属性定位 //tagName[@attrName=”attrValue”]
