一、MySQL数据类型

常用数据类型： ​ bit(只能是0/1) ​ varchar(可变长度字符串)，char定长字符串 ​ date（日期） time(时间) datetime（日期时间） ​ enum 枚举类型（规定了状态，只能是状态之一）

二、字段约束

常用约束： ​ auto_increment 自增约束 ​ primary key 主键约束（唯一，可用于判断数据是否唯一） ​ not null 非空约束（录入数据时必须有值） ​ unique 唯一约束（不得与任何其他数据重复，比如身份证号码） ​ default 默认约束（即未写入状态下的默认取值） ​ auto_increment 自增
